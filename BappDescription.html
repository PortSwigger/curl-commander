CURL Commander converts clipboard-copied curl commands directly into HTTP requests within Repeater, eliminating manual request reconstruction. The extension parses curl commands from your clipboard and automatically creates properly formatted requests in new Repeater tabs.

Features

  • One-click conversion from clipboard curl commands
  • Supports all HTTP methods and request bodies
  • Handles headers, authentication, and complex URLs
  • Works with multiline curl commands
  • Automatic Repeater tab creation

Usage

  1. Copy any curl command to your clipboard:
curl -X POST https://api.example.com/users \
  -H "Authorization: Bearer token123" \
  -H "Content-Type: application/json" \
  -d '{"name": "John", "email": "john@example.com"}'
  1. Right-click anywhere in the interface and select Send CURL to Repeater
  2. The parsed request automatically opens in a new Repeater tab with all headers, method, and body data preserved
